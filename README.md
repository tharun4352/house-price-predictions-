Machine Learning project for predicting house prices using Random Forest Regression.

## Problem Statement
Predict house sale prices using machine learning.

## Dataset
Housing sales dataset containing property features and sale prices.

## Approach
- Data Cleaning
- Missing Value Imputation
- One-Hot Encoding
- Random Forest Regressor

## Files
- Housing_Price_Predictor.ipynb
- train_predictions.csv
- test_predictions.csv
- model.pkl

## Technologies
- Python
- Pandas
- Scikit-Learn
- Google Colab
